# marten_recipes

Public guest-facing site for the Marten family recipe collection —
deployed via GitHub Pages. Guests see only recipe titles and ingredient
lists here, plus a favorites voting leaderboard.

This repo only ever contains generated, guest-safe data. The full
recipe collection (steps, photos, private tracking) lives in the
private [marten-recipes-data](https://github.com/martenrobert/marten-recipes-data)
repo and is never published here.
